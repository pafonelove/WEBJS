JavaScript — это язык программирования, который используют для написания frontend- и backend-частей сайтов, а также мобильных приложений.
JavaScript — это лучший друг HTML и CSS.
HTML задает разметку сайта, CSS отвечает за внешний вид, а JavaScript все это оживляет.
Сейчас JavaScript — единственный язык программирования для браузеров.
Он работает под Windows, macOS, Linux и на мобильных платформах, то есть везде.
В JS можно работать с переменными, объектами, функциями, условиями, математическими и логическими операторами и так далее.
Давайте разберем это на примере реальных возможностей.
